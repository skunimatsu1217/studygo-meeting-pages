🎯 ダイナミックプライシング戦略

基本方針：「3ヶ月の壁」を超えさせるために、初期コストを下げて入会ハードルを下げ、段階的に通常料金に戻す。
早く申し込むほどお得 = 「早期申込インセンティブ」で申込を前倒しさせる。

割引率の決定ロジック

💡 考え方

月額料金 = ¥9,900（税込）の場合

① 早期申込ボーナス: 開業の何ヶ月前に契約したか
  3ヶ月前 → 最大割引 / 2ヶ月前 → 中割引 / 1ヶ月前 → 小割引 / 当月 → 通常

② 段階的回復: 割引は月ごとに減少（徐々に通常料金へ）
  初月: 最大割引 → 翌月: 中割引 → 翌々月: 小割引 → 4ヶ月目: 通常料金

③ 損益分岐: 3ヶ月利用で会社側の獲得コストを回収
  割引総額 < 3ヶ月分通常料金 × 粗利率
  割引総額 < ¥29,700 × 0.6 = ¥17,820 以内に収める

料金計算ロジック（変更箇所）

// ShopContractPriceService.ts

// 1. クーポンにルールがあるか確認
rules = CouponDiscountRule.findBy(couponId)

// 2. 各月の料金に割引適用
for (month in [初月, 翌月, 翌々月]) {
 rule = rules.find(monthOffset == month)
 if (rule) {
  price = price * (1 - rule.discountRate / 100)
 }
}

// 3. 月次課金バッチでも同様に参照
monthsSinceStart = diff(now, contract.startDate)
rule = rules.find(monthOffset == monthsSinceStart)
chargeAmount = rule ? price * (1 - rate/100) : price

📝 まとめ：なぜこの設計が有効か

🔴 課題（現状）

・平均76日（3ヶ月未満）で解約
・短期で離脱するユーザーが多い
・クーポンは入会金にしか効かない
・「3ヶ月の壁」を超えられない

🟠 解決策

・初期コスト¥0で入会ハードル撤廃
・3ヶ月かけて段階的に通常料金へ
・早期申込ほどお得（焦燥感）
・「無料だから試してみよう」を促進

🟢 期待効果

・3ヶ月継続率: 17% → 40%+
・平均LTV: ¥15,400 → ¥28,600
・利用習慣の定着
・口コミ・紹介の増加
