1. 背景・目的

1.1 現状の課題

指標値問題点
契約継続期間 平均76日（約2.5ヶ月）3ヶ月未満で離脱が多い
最長継続期間145日（約4.8ヶ月）長期ユーザーが少ない
現行クーポン適用範囲入会金のみ月額料金・店舗初期費用への割引ができない

1.2 改修の目的

🎯 目的1：3ヶ月継続率の改善

初期コストを下げて入会ハードルを撤廃し、段階的に通常料金に戻すことで、「3ヶ月の壁」を超えさせる。

🎯 目的2：早期申込のインセンティブ

開業前の先行入会申込に対して、早く申し込むほどお得な割引を提供する（ダイナミックプライシング）。

🎯 目的3：多様な料金プランへの対応

月額プランだけでなく、ドロップイン・期間パスにも割引を適用可能な汎用的な仕組みを構築する。

1.3 想定する割引パターン

4.2 新規テーブル: CouponDiscountRule

カラム名型NULL制約説明
idIntNOT NULLPK, AUTO_INCREMENT主キー
couponIdIntNOT NULLFK → Coupon.id親クーポン
monthOffsetIntNOT NULL0〜50=初月, 1=翌月, ...5=6ヶ月目
discountTypeStringNOT NULLDEFAULT "rate""rate"=割引率(%) / "amount"=定額割引(円)
discountRateIntNOT NULL0〜100割引率(%) 100=無料（discountType="rate"時）
discountAmountInt?NULL0以上定額割引(円)（discountType="amount"時）
targetTypeStringNOT NULLenum的下記参照
createdAtDateTimeNOT NULLDEFAULT now()
updatedAtDateTimeNOT NULL@updatedAt
UNIQUE制約: (couponId, monthOffset, targetType) 同一クーポン × 同一月 × 同一対象タイプ の重複登録を防ぐ。

discountType の定義値

値説明計算方法例
rate割引率（%）Math.round(price * (1 - discountRate / 100))discountRate=50 → 半額
amount定額割引（円）Math.max(0, price - discountAmount)discountAmount=1000 → 1,000円引き

targetType の定義値

入会金（REGISTRATION_FEE = ¥5,500）と店舗別初期費用（shop.initialCost）は別の料金項目です
  • 入会金: 全店舗共通の固定金額（¥5,500）。初回契約時のみ。USER_PAYMENT_DETAIL_TYPE.ENTRY_FEE
  • 店舗別初期費用: 店舗ごとに設定される可変金額（shop.initialCost）。USER_PAYMENT_DETAIL_TYPE.SHOP_INITIAL_FEE
クーポンではそれぞれを個別に割引対象として指定できます。
値説明適用箇所料金ソース
initial_fee入会金初回契約時の入会金に適用REGISTRATION_FEE (¥5,500固定)
shop_initial_fee店舗別初期費用店舗別の初期費用に適用shop.initialCost (店舗ごとに異なる)
monthly月額プラン料金月額プランの各月の料金に適用shopPlan.price
dropinドロップイン利用料ドロップイン利用料に適用shopPlan.price (DROPIN型)
period_pass期間パス料金期間パス料金に適用（将来対応）-

4.3 Prisma Schema 追記内容

// prisma/schema.prisma に追加

model CouponDiscountRule {
  id             Int      @id @default(autoincrement())
  coupon         Coupon   @relation(fields: [couponId], references: [id])
  couponId       Int
  monthOffset    Int      // 0=初月, 1=翌月, ...5=6ヶ月目
  discountType   String   @default("rate")  // "rate" | "amount"
  discountRate   Int      // 割引率(%) 0〜100（discountType="rate"時）
  discountAmount Int?     // 定額割引(円)（discountType="amount"時）
  targetType     String   // "monthly" | "dropin" | "period_pass" | "initial_fee" | "shop_initial_fee"
  createdAt      DateTime @default(now()) @map("created_at") @db.Timestamp(0)
  updatedAt      DateTime @updatedAt @map("updated_at")

  @@unique([couponId, monthOffset, targetType])
}

  // ... 既存フィールド ...

  // ... 既存フィールド ...
  couponDiscountRulesJson String? @db.Text  // ← 追加（JSON文字列でスナップショット）

Input（変更後）

// definition.ts - adminSaveCoupon
input: {
  id?: number
  isActive: number
  name: string
  code: string

  validTo: string

    discountAmount?: number   // 定額割引(円)（discountType="amount"時）

処理フロー

// adminCouponResolver.ts - adminSaveCoupon 改修

// 1. 既存のCoupon保存処理（変更なし）
await prisma.coupon.upsert({ ... });

// 2. 割引ルールの洗い替え（新規追加）
if (discountRules) {
  // 既存ルールを全削除
  await prisma.couponDiscountRule.deleteMany({
    where: { couponId: targetId }
  });
  // 新ルールを一括登録
  await prisma.couponDiscountRule.createMany({
    data: discountRules.map(rule => ({
      couponId: targetId,
      monthOffset: rule.monthOffset,
      discountType: rule.discountType ?? 'rate',
      discountRate: rule.discountRate,
      discountAmount: rule.discountAmount ?? null,
      targetType: rule.targetType,
    }))
  });
}

重要: ハイブリッド決済フロー
  • 初回決済（PaymentIntent）: 初月（日割り）+ 翌月 = 2ヶ月分
  • 3ヶ月目以降（Subscription）: 毎月15日に翌月分を自動課金
アプリの契約確認画面には初回決済の2ヶ月分のみを料金内訳として表示する。
3ヶ月目以降の割引情報は、マイページ・決済履歴で確認できるようにする。

8.2 割引適用の共通関数

// 割引計算の共通関数（rate / amount 両対応）
function applyDiscount(originalPrice: number, rule: CouponDiscountRule): number {
  if (rule.discountType === 'amount') {
    // 定額割引: 0円を下回らないようにする
    return Math.max(0, originalPrice - (rule.discountAmount ?? 0));
  }
  // 割引率: 従来通り
  return Math.round(originalPrice * (1 - rule.discountRate / 100));
}

8.3 初回契約時の料金計算（ShopContractPriceService.ts）

getPrices: (

  coupon,
  couponDiscountRules: CouponDiscountRule[],  // ← 追加
  isThe1stContract, now
) => {
  // ======== 入会金（REGISTRATION_FEE = ¥5,500 固定） ========
  if (isThe1stContract) {
    let registrationFee = REGISTRATION_FEE;

    // 新: CouponDiscountRuleに initial_fee ルールがあれば優先

      r => r.targetType === 'initial_fee' && r.monthOffset === 0
    );
    if (initialFeeRule) {
      registrationFee = applyDiscount(registrationFee, initialFeeRule);
    } else if (coupon) {
      // 後方互換: 既存のdiscountRateで入会金割引
      registrationFee = Math.round(registrationFee * (1 - coupon.discountRate / 100));
    }
    prices.push({ label: '入会金', price: registrationFee, originalPrice: REGISTRATION_FEE });
  }

  // ======== 初月（日割り） ========
  let firstMonthPrice = getMonthlyPricePerDiem(dateStart, basePrice);
  const month0Rule = couponDiscountRules.find(
    r => r.targetType === 'monthly' && r.monthOffset === 0
  );
  if (month0Rule) {
    const originalPrice = firstMonthPrice;
    firstMonthPrice = applyDiscount(firstMonthPrice, month0Rule);
    prices.push({
      label: `${month}月分（日割り）`,
      price: firstMonthPrice,
      originalPrice,
      discountRate: month0Rule.discountRate,
    });
  }

  // ======== 翌月 ========
  let nextMonthPrice = basePrice;
  const month1Rule = couponDiscountRules.find(
    r => r.targetType === 'monthly' && r.monthOffset === 1
  );
  if (month1Rule) {
    const originalPrice = nextMonthPrice;
    nextMonthPrice = applyDiscount(nextMonthPrice, month1Rule);
    prices.push({
      label: `${nextMonth}月分`,
      price: nextMonthPrice,
      originalPrice,
      discountRate: month1Rule.discountRate,
    });
  }

  // ======== 店舗別初期費用（shop.initialCost） ========
  const isDefaultPlan = !!shopPlan.maxMinutes;
  const shopInitialCost =
    shop.initialCost && (shopPlan.withInitialFee || isDefaultPlan) ? shop.initialCost : 0;

  if (shopInitialCost > 0) {
    let discountedInitialCost = shopInitialCost;
    const shopInitialFeeRule = couponDiscountRules.find(
      r => r.targetType === 'shop_initial_fee' && r.monthOffset === 0
    );
    if (shopInitialFeeRule) {
      discountedInitialCost = applyDiscount(shopInitialCost, shopInitialFeeRule);
    }

      label: '店舗別初期費用',
      price: discountedInitialCost,
      originalPrice: shopInitialCost,
    });

      amount: discountedInitialCost,
      note: '店舗別初期費用',
    });
  }

8.4 月次課金バッチの変更（BatchContractService）

重要: 現在の月次課金バッチではクーポン割引が一切参照されていない Stripeサブスクリプション（毎月15日）で shopPlan.price をそのまま課金しているため、3ヶ月目以降の割引を適用するにはバッチの改修が必須。
※ 初月・翌月の2ヶ月分は初回決済（PaymentIntent）で処理されるため、バッチ改修の対象は3ヶ月目以降のみ。 
// BatchContractService - renewMonthlyContracts() の変更

// 1. 契約に紐づくクーポン割引ルールを取得
const couponHas = await prisma.shopContractHasCoupon.findFirst({
  where: { shopContractId: contract.id }
});

let chargePrice = shopPlan.price;  // デフォルト: 通常料金

if (couponHas?.couponDiscountRulesJson) {
  const rules = JSON.parse(couponHas.couponDiscountRulesJson);

  // 2. 契約開始月からの経過月数を計算
  const monthsSinceStart = dayjs(now)
    .diff(dayjs(contract.startDate), 'month');

    r => r.targetType === 'monthly' && r.monthOffset === monthsSinceStart

  // 4. ルールがあれば割引適用（rate / amount 両対応）
  if (rule) {
