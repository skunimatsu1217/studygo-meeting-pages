新機能 画面設計イメージ

クーポン利用履歴機能

加盟法人（会社管理画面）からクーポンの利用実績を確認できる機能

背景 現状の課題と新機能の目的

加盟法人がクーポンの利用状況を把握できない問題を解決

現状（Before）

  • 店舗一覧に「先月のクーポン利用枚数」の数字のみ表示
  • どのクーポンが使われたか分からない
  • 誰が使ったか分からない
  • いつ使われたか分からない
  • 割引額の合計が把握できない

新機能（After）

  • クーポン利用履歴一覧で全利用記録を確認
  • クーポン名・コード・割引率を表示
  • 利用者（会員名）を表示
  • 利用日時・契約店舗を表示
  • 統計サマリーで割引額の合計を把握
  • 店舗別・期間別のフィルタリング
  • CSV エクスポート対応

実装に必要な作業一覧

この画面設計を実現するために必要なバックエンド・フロントエンドの開発項目

# 作業項目 レイヤー 種別 概要
1 クーポン利用履歴 API API 新規 companyCouponHistory リゾルバーの作成。ShopContractHasCoupon を店舗・期間で検索
2 クーポン利用詳細 API API 新規 companyCouponHistoryDetail リゾルバーの作成。契約・料金情報を含む
3 統計サマリー API API 新規 利用数・割引合計の集計エンドポイント
4 クーポン利用履歴ページ フロント 新規 /alliance/coupon-history ページ作成（一覧 + フィルタ + 統計）
5 クーポン利用詳細ページ フロント 新規 /alliance/coupon-history/[id] ページ作成
6 サイドメニュー追加 フロント 改修 サイドナビに「クーポン利用履歴」メニュー追加
7 契約申請一覧にクーポン列 フロント 改修 contractApplications の config にクーポン列追加
8 店舗一覧に累計・リンク フロント 改修 shops の config に累計列・履歴リンク追加
